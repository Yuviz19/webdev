# Minimal Firefox Startpage

A lightweight, custom startpage built with plain HTML, CSS, and JavaScript.

Designed to be fast, distraction-free, and keyboard-friendly.

---

## Features

* Search directly from the homepage
* Custom link manager (add/delete)
* Persistent links using localStorage
* Weather display (wttr.in)
* Minimal, centered UI
* Keyboard-focused workflow

---

## Setup

Clone the repo and open `index.html` in your browser.

```bash
git clone <your-repo-url>
cd <repo-folder>
```

---

## Usage

* Type and press **Enter** to search
* Add or remove links via buttons
* Links persist across sessions
