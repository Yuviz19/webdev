# DISPLAY PROPERTIES

- determines how an element behaves on the screen
- the 4 horse-men of display properties

1. block
  - takes up the full width available
  - starts on a new line
  - any element after these are forced to start to the next line
  - div, p, heading tags, ul, section

2. inline
  - takes width as required by the content's need
  - does not start on a new line
  - it ignores the manual height and width properties
  - span, a, strong, em

3. inline-block
  - it is a hybrid values b/w inline and block
  - it does not starts on a new line (side by side fitting)
  - it fully respects manual height, width, padding and margins 

4. none
  - it completely removes the item from the webpage
  - the browser acts if the element does'nt exists anymore 
  - it's occupied space completely collapses
